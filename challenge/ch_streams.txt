For a given class domain, calculate average salary using Java 8 streams.

Class domain:
public class Person {
    private String name;
    private int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public Integer getAgeInteger() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }
}
